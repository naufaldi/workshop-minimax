Syntax Highlighting

Code examples are beautifully highlighted

JavaScript


// Fibonacci sequence
function fib(n) {
  if (n <= 1) return n;
  return fib(n-1) + fib(n-2);
}

console.log(fib(10)); // 55

Python


# List comprehension
squares = [x**2 for x in range(10)]

# Generator expression
gen = (x**2 for x in range(10))

    return f"Hello, {name}!"

Line Highlighting

Highlight specific lines in your code


function calculateFactorial(n) {
  // Base case: 0! = 1
  if (n === 0) return 1;

  // Recursive case
  return n * calculateFactorial(n - 1);
}

console.log(calculateFactorial(5)); // 120

Use data-line-numbers="2|4-6" to highlight specific lines

Transition Effects

Multiple transition styles available

  • none - No transition
  • fade - Cross fade
  • slide - Slide in/out
  • convex - Convex slide
  • concave - Concave slide
  • zoom - Zoom in/out

Set in Reveal.initialize({ transition: 'slide' })

Configuration Options


Reveal.initialize({
  // Display
  width: 960,
  height: 700,
  center: true,

  // Navigation
  controls: true,
  progress: true,
  slideNumber: true,
  hash: true,

  // Transitions
  transition: 'slide',
  transitionSpeed: 'default',

  // Plugins
  plugins: [
    RevealMarkdown,
    RevealHighlight,
    RevealNotes
  ]
});
